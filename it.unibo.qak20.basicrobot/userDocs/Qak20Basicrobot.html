Project

The basicrobot

basicrobot.qak in project it.unibo.qak20.basicrobot 		The 'basicrobot'
  • is able to execute basic move commands, by handling a dispatch (for the elementary commands w | s | ... ) and a request (for the command step);

  • maintains information on its current functional state;

  • interacts with a concrete robot by means of a (POJO) object (resources/unibo/robot/robotSupport.kt in project it.unibo.qak20.basicrobot) working as an adapter for devices described in the configuration file basicrobotConfig.json ;

  • is a source of collision events, that are generated by the low-level robot support
